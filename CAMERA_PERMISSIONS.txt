STEP 1: Check Camera Device
   ls -l /dev/video*
   
   Should show: /dev/video0, /dev/video1, etc.

STEP 2: Add User to Video Group
   sudo usermod -a -G video $USER
   
   Then LOG OUT and LOG BACK IN (important!)

STEP 3: Set Permissions
   sudo chmod 666 /dev/video0

STEP 4: Install v4l-utils
   # Ubuntu/Debian
   sudo apt-get install v4l-utils
   
   # Check camera
   v4l2-ctl --list-devices

STEP 5: Test Camera
   # Install cheese (camera viewer)
   sudo apt-get install cheese
   cheese
   
   If cheese works, Python should work too

COMMON LINUX ERRORS:

Error: "Permission denied"
Fix: Run step 2 (add to video group) and RESTART

Error: "/dev/video0: No such file"
Fix: Camera not detected - check USB connection
